# PCB下单指南

本指南以嘉立创下单助手（桌面端）为例，介绍如何下单 PCB。

## 下单步骤

### 1. 进入 PCB 下单页面

进入主页 → 点击 **PCB下单**

![进入PCB下单](/images/1_1.png)

### 2. 上传 Gerber 文件

点击 **上传PCB**

![上传PCB](/images/1_2.png)

### 3. 选择文件

选取你所下载的 Gerber 文件（请以实际文件名为准）

![选择Gerber文件](/images/1_3.png)

### 4. 提交订单

解析完成后点击 **立即下单**

![立即下单](/images/1_4.png)

## 参数选择

::: warning 注意
以下参数为默认情况，若版本文档有详细指明请使用版本文档中的数据
:::

| 步骤 | 选项名称 | 默认选项 | 备注 |
| --- | --- | --- | --- |
| 1 | 材料 | FR4 | / |
| 2 | 尺寸 | 自动识别 | 若没有成功识别请在交流群中询问 |
| 3 | 板子数量 | / | 随心所欲，建议留出几个备用 |
| 4 | 板子层数 | 2 | / |
| 5 | 确认生产稿 | 不需要 | / |
| 6 | 出货方式 | 单片 | / |
| 7 | 板上加标志 | / | 随心所欲，部分选项需要收费 |
| 8 | SMT贴片 | 不需要 | 除非你付得起SMT高昂的开机费用 |
| 9 | 钢网 | 不需要 | TB定做似乎更便宜 |

其他信息（开票、资料、快递）自行选择即可。

## 提交订单

点击提交订单后等待审核完成付款生产即可。
