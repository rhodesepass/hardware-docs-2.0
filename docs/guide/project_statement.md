后续通行证开发将以 Issue / Pull Request 的方式推进。功能建议请提交至：[issue tracker](https://github.com/rhodesepass/electric_pass/issues)

提交时请尽量说明：期望实现的效果、所属功能模块、使用场景/复现步骤（如有）。社区成员将根据兴趣选择认领与实现。
