## 如何成为贡献者

如果你想为本项目做出贡献，可以通过以下方式参与：

- **提交代码**：Fork 仓库并提交 Pull Request
- **报告问题**：在 [GitHub Issues](https://github.com/rhodesepass/docs/issues) 中反馈问题
- **改进文档**：帮助完善和翻译文档
- **分享项目**：向更多人介绍这个项目

加入 [rhodesepass](https://github.com/rhodesepass) 组织，成为公开成员即可出现在此名单中。
