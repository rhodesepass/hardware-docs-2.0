# 调试概述

本章节介绍调试和排障相关内容。

## 调试工具

| 工具 | 用途 |
| --- | --- |
| 万用表 | 测量电压、检查导通性 |
| USB转串口 | 在屏幕不工作时调试设备 |
| 放大镜 | 检查焊点质量 |

## 常用调试方法

### 电压测量

使用万用表测量各路供电电压，确保在正常范围内。

### 导通性检查

检查关键信号线是否导通，排查虚焊问题。

### 串口调试

通过 USB 转串口工具连接设备，查看系统启动日志。
