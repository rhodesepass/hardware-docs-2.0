# 常见故障

本文档汇总常见故障及解决方法。

## 电源相关

### 供电电压异常

**症状**：三路供电没有全部出完或电压不对

**可能原因**：
- EA3036 虚焊
- 反馈脚接触不良

**解决方法**：补焊或更换 EA3036

### 上电大电流

**症状**：上电直接大电流，电源灯不亮

**可能原因**：
- 屏幕背光芯片焊接错误
- 二极管焊反

**解决方法**：检查屏幕背光区域和二极管方向

## USB 相关

### USB 报错 42/43

**症状**：Device failed to start

**可能原因**：
- USB Type-C 虚焊或连锡
- 主控 USB 引脚虚焊

**解决方法**：检查并补焊 USB 相关引脚

### 无法进入下载模式

**症状**：烧录中断后无法重新进入下载模式

**解决方法**：参考 [BiliBili 刷新教程](https://www.bilibili.com/video/BV1Xu28BDEjL/)

## 存储相关

### Flash 不识别

**症状**：提示 Flash 不支持

**可能原因**：NAND 区域虚焊

**解决方法**：检查并补焊 NAND 引脚

## 屏幕相关

### 屏幕不显示

**症状**：烧录完成后屏幕无显示

**排查步骤**：
1. 检查背光是否亮起
2. 检查 6 个关键信号点电压
3. 补焊问题引脚

### 颜色异常

**症状**：屏幕颜色显示错误

**可能原因**：颜色线虚焊

**解决方法**：补焊颜色信号线

### 亮度无法调节

**症状**：亮度一直最亮

**可能原因**：SCREEN_BL_CTRL 脚虚焊

**解决方法**：补焊亮度控制引脚
