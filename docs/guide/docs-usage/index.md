## 添加新文档

添加新说明书时，请同步更新本目录：

1. 在 `docs/guide/docs-usage/` 目录下创建新的 `.md` 文件
2. 在 `docs/.vitepress/config.mts` 的侧边栏配置中添加链接
3. 在上方「文档列表」表格中添加一行

::: tip 文件命名规范
- 使用小写字母和连字符：`my-document.md`
- 文件名应简洁明了，反映文档内容
:::
