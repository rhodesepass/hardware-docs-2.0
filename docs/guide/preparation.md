# 准备工作

在开始复刻之前，请确保您已准备好必要的工具和文件。

## 你需要的工具

| 名称 | 型号 | 备注 |
| --- | --- | --- |
| 电烙铁 | T12 | 可控温，最便宜的那种就行，功率至少30W |
| 烙铁头 | 刀头 | 对自己好点，买个好使的 |
| 锡浆 | / | / |
| 焊锡丝 | / | / |
| 助焊剂 | / | / |
| 加热台 | / | 加热台一定要能控温 |
| 镊子 | / | 买三把的套件，自己看哪种合适 |
| 串口调试工具 | CH34X | 用于调试 |
| 万用表 | / | 调试用，买个最便宜的有电压档和导通挡的就行 |

## 下载所需文件

你可以在 Release 中下载所需文件：

[前往 Release 下载](https://github.com/inapp123/epass_hardware/releases)

### 你至少需要

- **BOM** - `electric_pass_baseboard_V#.#.#_.BOM.html`
- **PCB (Gerber)** - `electric_pass_baseboard_V0.3.1_.GERBER.zip`
- **Firmware** - 详见[固件部分](./firmware/)

## 技能要求

::: warning 重要提示
本项目包含 QFN 表面贴装器件，如果您此前没有使用表面贴装器件进行 DIY 项目的经验，建议您通过其他较简单的项目练习贴装，**不要直接开始制作**。
:::

建议您具备以下经验：
- 0603 及以上封装贴片元件的焊接经验
- 基本的电路调试能力
- 使用万用表测量电压和导通性
